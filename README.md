# Sentinel
Sentinel is a real-time surveillance intelligence platform that augments standard video streams with advanced AI-driven perception and decision layers. Built on modern computer vision and deep learning pipelines, Sentinel performs multi-class object detection, temporal event analysis, and context-aware filtering to deliver precise, low-latency security insights.

The system is designed to minimize false positives by combining spatial detection with behavioral inference, enabling reliable identification of meaningful events across dynamic environments.
